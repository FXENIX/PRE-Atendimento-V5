EVOLUTION API v2.0 - LISTA COMPLETA DE ENDPOINTS
================================================
Base URL = CAMPO > URL da Evolution API

Autenticação: Header = CAMPO > "API Key"

instance =  CAMPO > Instância na Evolution API

CAMPO EM BRNACO SERA PRENCHIDO POR > Criar instância



=== INSTANCE ===
POST   /instance/create
GET    /instance/fetchInstances
GET    /instance/connectionState/{instanceName}
GET    /instance/connect/{instanceName}
DELETE /instance/delete/{instanceName}
DELETE /instance/logout/{instanceName}
POST   /instance/restart/{instanceName}
POST   /instance/setPresence/{instanceName}

=== OUTROS ÚTEIS ===
POST   /message/sendText (com quotedMessage)
POST   /chat/findMessages (com filtros)
GET    /instance/fetchInstances (com status detalhado)

Observações v2.0:
- A maioria dos endpoints usa {instanceName} no path
- Envio de mídia usa multipart/form-data
- Respostas geralmente seguem padrão { "status": "success", "data": {...} }
- Autenticação via header "apikey"
